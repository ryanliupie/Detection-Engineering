# Threat Hunting 

## What is Threat Hunting?

Threat hunting is the <b>proactive</b> approach to search for cyber threats that are lurking undetected in a network guided by <b>threat intelligence</b>. This process digs deep to find adversaries within an environment that have bypassed initial network defences. 

Attackers may stealthily remain to gather data, or obtain credentials that would allow them to move laterally across the environment. Many organizations lack the capabilities to stop advanced persistent threats, and so threat hunting is an essential component when it comes to defence.
